# 🚀 Narun J.S. — Interactive Developer Portfolio

An interactive 3D developer portfolio and digital showcase built with **React**, **TypeScript**, **Vite**, **TailwindCSS**, **Framer Motion**, and **Spline 3D**.

![Narun Portfolio Logo](/public/logo.png)

## ✨ Features

- **Interactive 3D Experiences**: Spline 3D robot model, interactive canvas particle effects, and 3D card tilt physics.
- **Ultra-Smooth Theme Switching**: Circular View Transitions API with liquid fallback animation for 60 FPS dark/light mode toggling.
- **Interactive Certificate Viewer**: Full-screen modal certificate preview with embedded PDF viewers and verified credential badges.
- **Fluid Custom Cursor**: Smooth spring-based custom cursor tracking with fallback on interactive preview elements.
- **Responsive Layout**: Designed for mobile, tablet, and desktop viewports.

---

## 🛠️ Tech Stack

- **Frontend Framework**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS + Custom Vanilla CSS Variables
- **Animations**: Framer Motion + HTML5 Canvas API + View Transitions API
- **3D Assets**: Spline 3D Scene Integration
- **Icons**: Lucide React

## 💻 Getting Started Locally

### Prerequisites

Ensure you have **Node.js 18+** and **npm** installed on your system.

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/narunsenthilkumar/portfolio.git
cd portfolio

# 2. Install dependencies
npm install

# 3. Start the local development server
npm run dev
```

Open your browser at `http://localhost:5173`.

---

## 📦 Production Build

```bash
# Build for production with TypeScript compilation
npm run build

# Preview production build locally
npm run preview
```

## 🌐 Deploying to Vercel

This project includes a pre-configured `vercel.json` file.

1. Push your repository to **GitHub**.
2. Go to [Vercel Dashboard](https://vercel.com/dashboard) and click **"Add New Project"**.
3. Import your GitHub repository.
4. Keep default settings:
   - **Framework Preset**: Vite
   - **Build Command**: `npm run build`
   - **Output Directory**: `dist`
5. Click **Deploy**.
